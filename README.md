# GU Rocketry

Repository for work completed as part of GU Rocketry.

## Structure

- `recovery-systems/parachute-area/`
  - `parachute_area.py`: command-line parachute sizing calculator
  - `parachute_gui.py`: Qt-based GUI for the same sizing workflow
  - `requirements.txt`: Python dependencies for the parachute tool

## Repository Notes

- The top-level `README.md`, `LICENSE`, and `.gitignore` are the authoritative repository metadata.
- Local virtual environments such as `.venv/` are ignored and are not part of the tracked source tree.
